# IRREAC Output Format

## Issue
Whether [legal question] given [key facts]. (One sentence. Framed as a yes/no question.)

## Rule Statement
[Specific rule with citation — Restatement § X, UCC § X-XXX, statute name, or landmark case name + year.]

## Rule Exploration
[How courts have interpreted this rule. Key cases. Majority vs. minority jurisdictions. Any notable circuit split or Restatement vs. common law divergence.]

## Application

### Element 1: [Element Name]
[Analysis of how facts satisfy or fail this element.]
[Counter-argument: strongest opposing position + why it does/does not succeed.]

### Element 2: [Element Name]
[Analysis...]

### Element N: [Element Name]
[Analysis...]

## Conclusion
[Direct answer to the Issue. Confidence: High/Moderate/Low. One sentence explaining the confidence level.]
